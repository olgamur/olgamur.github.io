Правила игры

Перемещая соты с номерами по одной, необходимо выстроить их таким образом, чтобы числа располагались по порядку возрастания в направлении слева вверх. Т.е. левее всех должна быть сота с номером 1, выше всех — с номером 4, ниже всех — с номером 13, и правее всех — пустая ячейка. На каждом шаге можно переместить одну из сот, окружающих пустую ячейку (максимум их 6, минимум 2).
